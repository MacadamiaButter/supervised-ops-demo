# Amenities & common areas
Fitness rooms are keyfob-access, 6am-11pm daily. Pool (seasonal
properties only) is open Memorial Day-Labor Day, 9am-9pm, no lifeguard --
swim at own risk, children under 14 must be accompanied. Community
rooms can be reserved for private events through the portal, 48-hour
notice, cleaning deposit $100 refundable. Package lockers are at each
building's main entrance; carriers deliver directly, tenants get a code
by text. Common-area maintenance issues should be reported the same way
as unit maintenance requests.
